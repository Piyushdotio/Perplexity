Why Morgan is Used
1. Request Logging (Main Purpose)

Morgan logs details about every HTTP request hitting your server.

Example log:

GET /api/users 200 15ms - 512b

This tells you:

Method → GET
Route → /api/users
Status → 200
Response time → 15ms
Response size → 512 bytes

👉 Without logging, debugging backend becomes blind.
